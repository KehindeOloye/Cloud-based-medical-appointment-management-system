UGR MEDICAL CENTRE APPOINTMENT MANAGER

The UGR Appointment manager is an easy to use software with features for Patients and features for GP. It has the following pages;
The index.html: This is the welcome page. It is also the page with links to both patientappointment.html and gpappointment.html pages.
The patientappointment.html: This is a page that has a listing of all Scheduled Appointments, By entering his/her name, a patient can see all scheduled appointments. This includes all past and future ones. A patient (with his/her name entered already) can make an appointment. A date and time-slot must be specified. A patient can optionally specify a GP. If unspecified, the system will randomly assign an available GP in the requested time-slot.The patient can select an appointment from the list and delete it. By selecting an appointment from the list, the patient can update its details (e.g. changing the GP, or the time-slot). An appointment update goes through the same check as in creating an appointment (e.g. GP availability)
The gpappointment.html: The GP can see the Calendar by entering his/her name and selecting a year and month, a GP can see a monthly calendar display, showing whether a day has any appointment scheduled. Also, by selecting a date in the monthly calendar view, the GP can see a list of scheduled appointments on that day.
